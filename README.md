# AI-Recycling-Assistant 🚮🤖  
> **사진 한 장으로 끝내는 분리배출 가이드**  
> Gemini 기반 이미지 분석 + FAQ + 커뮤니티를 담은 재활용 도우미 서비스

---

## 📌 프로젝트 소개

**AI-Recycling-Assistant** 는  
쓰레기 분리배출 방법이 헷갈리는 사용자를 위해,

- 사진만 찍으면 **어떤 쓰레기인지 인식**하고  
- **어떻게 버려야 하는지**까지 안내해 주며  
- FAQ와 커뮤니티를 통해 **서로 정보를 나눌 수 있는**  
웹 기반 분리배출 안내 서비스입니다.

수도권 매립지 포화, 재활용률 저하, 잘못된 분리배출 문제를 완화하는 데
작게나마 기여할 수 있는 **실사용 가능한 MVP**를 목표로 개발되었습니다.

---

## ✨ 주요 기능 (Features)

### 1. AI 사진 분석 기능 📷

- 사용자가 쓰레기 사진을 업로드하면  
  → 백엔드에서 **Gemini 기반 AI**로 이미지를 분석  
  → 인식된 품목과 함께 **분리배출 방법 안내 텍스트**를 반환
- 목표 응답 시간: **5초 이내**
- 잘못된 분류를 줄이기 위해 **텍스트 설명 + 예시 안내**를 함께 제공

---

### 2. FAQ 빠른 가이드 📚

- 자주 헷갈리는 품목들을 **카테고리별 FAQ** 형태로 정리  
  - 예: 일회용 컵, 플라스틱 용기, 비닐류, 종이팩 등
- 검색/탭 이동을 통해 **빠르게 분리배출 방법을 확인**할 수 있도록 구성
- 추후 **지역별 기준 반영**을 고려한 구조로 설계 (확장 가능)

---

### 3. 커뮤니티 Q&A 기능 💬

- 사용자끼리 **분리배출에 대한 질문과 답변을 주고받을 수 있는 게시판** 제공
- “이건 어떻게 버려야 할까요?” 같은 상황을 다른 사용자와 공유하고 해결
- 유용한 답변은 상단에 노출될 수 있도록 향후 **추천·정렬 기능** 확장 가능

---

### 4. 계정 및 기본 서비스 기능 👤

- **회원가입 / 로그인 기능** 지원
- 로그인한 사용자는
  - AI 분석 요청
  - 질문 작성 및 댓글 작성
  - FAQ/커뮤니티 열람  
  등의 기능을 사용할 수 있는 구조로 설계

---

## 🧱 기술 스택 (Tech Stack)

> 실제 프로젝트에 맞게 수정해서 사용하면 됩니다.

### Frontend (Web)

- React Native 기반 웹 (Expo Web 등) 또는 React Web
- 주요 역할
  - 이미지 업로드 UI
  - 분석 결과 및 분리배출 안내 화면
  - FAQ / 커뮤니티 목록 & 상세 페이지
  - 로그인 / 회원가입 폼

### Backend

- REST API 서버
- 주요 역할
  - 이미지 업로드/전송 처리
  - Gemini API 호출 및 응답 가공
  - 사용자 인증/인가 처리
  - FAQ 및 커뮤니티 데이터 제공

### Database

- MySQL DB
- 저장 데이터
  - 사용자 정보
  - 커뮤니티 게시글 및 댓글
  - FAQ/분리배출 규칙 데이터

### AI

- **Gemini 기반 이미지 분석 모델** 사용
  - 입력: 쓰레기 이미지
  - 출력: 인식된 품목 + 분리배출 방법 설명

---

## 🏛 아키텍처 개요

1. 사용자가 웹 화면에서 **이미지 업로드 / 질문 작성 / FAQ 열람** 요청
2. Frontend → Backend로 API 요청 전송
3. Backend는
   - AI 분석이 필요하면 **Gemini API 호출**
   - DB에서 FAQ/커뮤니티/유저 정보 조회
4. 처리 결과를 다시 Frontend로 전달
5. Frontend에서 결과를 UI로 렌더링하여 사용자에게 표시

```text
[User] 
   ↓
[Frontend (Web)]
   ↓  REST API
[Backend Server] ──↔── [DB]
   ↓
[Gemini AI API]
